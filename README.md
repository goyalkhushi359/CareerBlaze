# CareerBlaze 🔥

AI-powered interview preparation platform that analyzes your 
body language, speech patterns, and answers in real-time — 
giving you a complete score and personalized feedback after 
every mock interview.

## Features
- 🎙️ Real-time fumble & filler word detection
- 🧍 Body language & posture scoring via MediaPipe
- 💼 Job fit % score using NLP
- 🛤️ Personalized career path roadmap
- 📊 Analytics dashboard with progress tracking

## Tech Stack
React.js · Python Flask · OpenCV · MediaPipe · Groq (Llama 3) · MySQL
